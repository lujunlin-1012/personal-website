Project 04 / AI Coding / Mini Program

Vibe Coding 宿舍分摊记账小程序

这是一个面向宿舍日常协作的小程序作品，围绕采购记账、费用分摊、值日安排、贡献榜和月度账单展开。 项目使用 Claude Code 连接 DeepSeek 辅助开发，把真实生活问题快速转化成可体验的小程序原型。

WorkflowClaude Code + DeepSeek
Type微信小程序
Feature记账 / 分摊 / 值日

作品介绍

宿舍生活中经常会出现公共采购、费用平均分摊、值日提醒和成员贡献不清晰的问题。 这个小程序把这些场景整合成一个轻量工具：室友可以记录采购，查看月度账单，完成值日任务， 也能通过贡献榜看到每个人在宿舍事务中的参与情况。

功能亮点

  • 首页聚合本月支出、人均费用、采购笔数和最近采购。
  • 采购记录支持分类、金额、成员和日期展示，便于后续结算。
  • 账单页提供月度统计、分类统计、对比上月和分享账单入口。
  • 值日表记录每周任务与完成状态，贡献榜形成正向反馈。
